## 自动备份脚本（建议主人启用）

```bash
# D:/backup-ai-config.bat
# 每天 23:00 跑（用 Windows 任务计划）

@echo off
set BACKUP_DIR=D:\CloudBackup\ai-config-%date:~0,4%%date:~5,2%%date:~8,2%
mkdir %BACKUP_DIR%

# 备份关键配置
copy "C:\Users\hao\.claude\settings.json" %BACKUP_DIR%\
copy "C:\Users\hao\.claude.json" %BACKUP_DIR%\
copy "G:\CC-Switch-v3.15.0-Windows-Portable.bak.20260627_130921\APIkey.docx" %BACKUP_DIR%\
copy "C:\Users\Administrator\Downloads\openclaw0417.pem" %BACKUP_DIR%\

echo Done: %BACKUP_DIR%
```

## 异地备份建议（强烈推荐）

让 vault 推 2 个远端：

```bash
cd /d/ObsidianVault

# 加 GitHub 远端
git remote add github https://github.com/haowenzhuo/obsidian-vault.git

# 推 GitHub
git push github master
```

这样即使 Gitee 挂了，GitHub 还有完整 vault。
