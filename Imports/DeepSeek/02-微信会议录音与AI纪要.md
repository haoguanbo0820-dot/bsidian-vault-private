### 一、如何录音

**微信本身没有内置录音功能**，需要借助系统或第三方工具。

#### 📱 手机端
- **用自带录屏功能**（推荐）
  - **iPhone**：下拉控制中心 → 长按录屏按钮 → 开启**麦克风** → 开始录制。这样就能同时录下屏幕和声音。
  - **安卓**：多数品牌在下拉菜单里也有“屏幕录制”，同样需要开启麦克风权限录制系统声音。
- **注意事项**：务必提前告知参会者并取得同意，避免隐私或合规风险。

#### 💻 电脑端（Windows / Mac）
- **Windows**：按 `Win + G` 打开游戏栏，点击录制，或使用第三方软件如 Audacity（需配置音频驱动）。
- **Mac**：用 **QuickTime Player** → 文件 → 新建屏幕录制，选择内置麦克风。要同时录系统声音需安装虚拟声卡插件（如 **BlackHole**）。

💡 **最佳音质方案**：手机端用另一台设备（如录音笔、备用手机）靠近外放录音；电脑端使用会议耳机。这样能大幅减少回声和杂音。

关键是把录音转成文字，再提炼纪要。目前没有一步到位的原生功能，但流程很成熟。

**最快路径：微信录音 → 语音转文字工具 → AI总结**

#### 推荐工具与流程

| 步骤 | 推荐工具 | 具体操作 |
|:---|:---|:---|
| **1. 录音** | 手机录屏 / 录音笔 | 得到录音文件 |
| **2. 语音转文字** | **飞书妙记**（免费额度足）<br>**通义听悟**（实时字幕+总结）<br>**讯飞听见**（准确率高） | 网页上传录音文件，自动转写 |

### 第一步：录制清晰音频（准备“原料”）

录音的好坏，直接决定会议纪要的质量。电脑端最推荐的办法是**只录制会议声音，不录其他杂音**。

**默认情况**：直接用微信外放，再用系统录音，会把房间里的所有声音都录进去，效果很差。你需要一个虚拟声卡，把“微信的声音”直接传给录音软件。

#### 推荐方案：用 Audacity + 虚拟声卡
这是免费且效果极佳的搭配，设置一次，以后都管用。

1.  **安装虚拟声卡插件**
    -   **Windows**：下载 **VB-Cable**（免费），它会创建一个虚拟输入输出设备。
    -   **Mac**：下载 **BlackHole**（免费），功能相同。

2.  **配置声音路由**
    -   打开电脑的声音设置。
    -   把**播放设备（输出）**设为刚才装的虚拟声卡，比如 `VB-Cable Input`。
    -   这时你电脑上的所有声音（包括微信）都会流入这个虚拟通道。
    -   **注意：这样设置后你自己听不见声音了，需要马上操作下一步。**

3.  **用 Audacity 录制并实时监听**
    -   打开免费软件 **Audacity**。
    -   **录音设备**选 `VB-Cable Output` 或 `BlackHole`。
    -   **播放设备**选你真实的耳机/音响。
    -   点击 Audacity 里的 **“传输” -> “软件监听”** 并勾选，就能边听边录。
    -   点录音键开始，你的微信会议音频就会被纯净地录下来。

💡 **更省事的备用方案**：如果觉得以上麻烦，可以直接用耳机模式。你将**耳机插到带麦克风的手机上，靠近电脑扬声器录音**，虽然会录入一点环境音，但操作最简单。

### 一、录音环节：双机协作，保证清晰度

在电脑上参加语音会议时，想让录音清晰纯净，最简单有效的方法就是“物理分离”。

#### 🎯 推荐方案：手机外放 + 录音笔/AI工具实时转写

**操作步骤：**

1.  **电脑负责通话**：你在电脑上正常登录微信，参与多人语音会议。为了让录音更清楚，建议戴上耳机收听，这样电脑扬声器不外放，减少回声。
2.  **手机负责录音和转写**：准备你的手机，打开 **通义听悟 App**（或 **讯飞听见 App**）。
3.  **启动实时转写**：
    -   在 App 里选择“实时转写”功能。
    -   将手机放在电脑的耳机旁（如果戴耳机，摘下一边耳机贴着手机麦克风；如果用外放，就靠近扬声器）。
    -   App 会边收音边转文字，通话结束时即可生成完整的文字记录。

**这个方案的好处是：**
-   **不影响通话**：电脑端微信通话不受任何干扰。
-   **即时可得**：挂断电话，转写和初步AI总结就快出来了，不用事后上传。
-   **声音清晰**：用耳机收音能最大程度避免房间杂音。

#### 🔄 备用方案：电脑内录（仅当不方便用手机时）

如果必须只用电脑，和之前一样，需要 **VB-Cable (Win) / BlackHole (Mac)** 虚拟声卡，配合 Audacity 录制。但设置略复杂，且微信语音意外中断可能导致录混音失效，不如双机方案稳定。

在微信语音通话中，它的音频处理机制和Zoom/腾讯会议不同，设置起来需要一点技巧。请按以下步骤操作，我尽量说得简洁。

---

### 第一步：安装必要工具

你需要两个免费工具：

1.  **VB-Cable** (虚拟音频线)
    -   下载地址：`vb-audio.com/Cable`
    -   解压后，右键以管理员身份运行 `VBCABLE_Setup_x64.exe`。
    -   安装后电脑会出现一个新的播放和录音设备，就叫“VB-Audio Virtual Cable”。

2.  **Audacity** (音频录制软件)
    -   下载地址：`audacityteam.org`
    -   这是一个免费开源的录音软件，界面简洁。

---

### 第二步：配置 Windows 音频路由

这是核心。我们要让微信的声音通过虚拟线传给你听，同时传给 Audacity 录制；而你的麦克风声音也要传进虚拟线，这样录出来的文件才有你和其他人的对话。

**1. 打开 Windows 声音设置**
-   右下角任务栏，右键点击喇叭图标 → **声音**。
-   切换到 **“录制”** 选项卡。

**2. 设置你的真实麦克风为“侦听”状态**
-   在列表里找到你正在使用的真实麦克风（例如“Realtek 麦克风”），右键 → **属性**。
-   切换到 **“侦听”** 选项卡。
-   勾选 **“侦听此设备”**。
-   在下拉菜单中，选择 **“VB-Audio Virtual Cable”**。
-   点击确定。
-   **这一步的目的**：把你的麦克风声音，注入到虚拟线里，这样微信就能收到，同时录音软件也能录到你说话。

**3. 设定微信的输出**
-   打开微信 → 设置 → 通用设置 → 语音和视频。
-   扬声器：选择 **“VB-Audio Virtual Cable”**。
-   麦克风：选择 **你真实的麦克风**（不要选错了，刚才让你设置侦听，就是为了让麦克风声音能通过虚拟线传给别人）。
-   **此时，你会突然什么都听不见了。** 正常，因为微信把声音输出了虚拟线，而不是你的耳机。接下来我们要把声音引回来。

**4. 让你自己也能听见会议声音**
-   再次进入 **Windows 声音设置** → **“播放”** 选项卡。
-   找到 **“VB-Audio Virtual Cable”** 这个播放设备，右键 → **属性**。
-   切换到 **“侦听”** 选项卡。
-   勾选 **“侦听此设备”**。
-   在下拉菜单中，选择 **你的真实耳机/扬声器**。
-   点击确定。
-   **至此，你耳机里又能听到微信声音了，你的麦克风他们也听得到，一切恢复正常。**

---

### 第三步：设置 Audacity 进行录音

现在，所有的声音都汇聚在 `VB-Audio Virtual Cable` 这条线上。我们只需要让 Audacity 录下这条线里的所有声音。

-   打开 **Audacity**。
-   在软件主界面的音频设置区域（通常在左侧或左上），找到 **录音设备**。
-   选择 **“VB-Audio Virtual Cable” (或 Cable Output)**。
-   播放设备，选择 **你的真实耳机**，这样才能听见。
-   现在点红色的录音键，开始录制。
-   你会在波形图上看到一条完整的音频，里面包含了你和所有参会者的声音。

---

### 🤖 DeepSeek (12/12)

可以的，完全可以用 Windows 自带的**录音机（语音录音机）**来代替 Audacity，这样更简单。但需要先把“录音设备”指定为虚拟声卡 `VB-Cable`，否则它默认只会录你的麦克风，录不到对方的声音。

我们沿用之前的思路，只要把 Audacity 这一步换成录音机就行。具体操作分两步：

---

### 1. 把系统默认录音设备设为虚拟声卡

这样录音机、以及其他所有录音软件，都会直接抓取虚拟声卡里的声音。

1.  右键任务栏喇叭图标 → **声音设置**。
2.  往下找到并点击 **“更多声音设置”**（Win11 在“高级”里）。
3.  切换到 **“录制”** 选项卡。
4.  找到 **VB-Audio Virtual Cable**（或 Cable Output），右键点击它 → **设置为默认设备**。
    -   设置成功后，这个设备旁边会出现一个绿色对勾。
5.  同时，建议把你真实的麦克风 **禁用**，避免干扰。右键你的麦克风 → **禁用**。

> 注意：你的麦克风声音已经在第一步，通过“侦听”功能汇入虚拟声卡了，所以禁用它不会影响对方听到你说话，也能避免录音机录出双重声音。

---

### 2. 打开录音机，直接录制

1.  在开始菜单搜索并打开 **“录音机”**（Win10 叫“语音录音机”，Win11 叫“录音机”）。
2.  界面很简单，直接点中间的**麦克风图标**就开始录音。
3.  录完后，文件会自动保存在录音机里，右键可以 **“打开文件位置”**，就能找到录音文件（通常是 `.m4a` 格式）。
4.  这个文件可以直接上传到飞书妙记里生成纪要。

---

### ⚠️ 使用录音机的一个小提醒

和 Audacity 不同，录音机**没有独立的“播放设备”选择**。因此：

-   录音期间，你必须像之前设置的那样，在声音设置里把 `VB-Cable` 的播放端**开启“侦听”并选到你的真实耳机**，才能边录边听会议。如果听不到，去检查这一步就行。
-   录音机生成的音频文件，通常已足够清晰，可以放心用。

这样就用电脑自带软件搞定了，省去安装 Audacity 的步骤。你先试试，有任何问题随时问我。
