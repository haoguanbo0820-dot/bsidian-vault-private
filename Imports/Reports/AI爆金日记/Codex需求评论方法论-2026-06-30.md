## 评论抓取 SOP（每天跑）

### 步骤

1. **HN 主题抓评论**（用 urllib，最快）
   - 关键词：`codex security`, `vibe coding reality`, `llm reliability`, `claude code issues`
   - Algolia API: `https://hn.algolia.com/api/v1/search?query={topic}&tags=comment`
   - 过滤：comment 数 >= 50 + 真实评论 >= 10

2. **抖音评论抓取**（用 CDP）
   - 抓"AI 副业""Vibe Coding""Claude Code 装"等爆款视频前 10 个
   - 抓每个视频的评论前 50 条
   - 主人抖音已登录（CDP 模式跑得通）

3. **小红书评论抓取**（用 CDP）
   - 抓"AI 副业"搜索结果前 20 篇笔记
   - 抓每篇评论前 30 条

4. **公众号文章 + 评论**（公开网页）
   - 找 10 个 AI/搞钱公众号（刘润、stormzhang、池建强、caoz、warfalcon 等）
   - 抓最新 5 篇 + 精选评论 10 条

5. **过滤 + 分类**（关键步骤）
   - 真信号：评论里写"愿付费""求装""帮帮我""解决不了"
   - 假信号：评论里"👍""学习了""马克" = 噪音
   - 跨平台验证：3 个平台都说 = 真

6. **写报告**（推 vault）
   - 每日 1 篇 `AI 爆金日记-YYYY-MM-DD-需求.md`
   - 真实信号 1/2/3 + 抓取记录 + 铁律
