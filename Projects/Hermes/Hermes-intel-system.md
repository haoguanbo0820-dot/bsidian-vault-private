# Hermes 情报系统

> 微信聊天 → 结构化分析 → 日报/周报/月报

## 数据管道

```
微信群聊消息
  ↓ wechat_data.db (13 张表，SQLite)
  ↓ 每日 08:00 增量扫描
  ↓ 问题检测 + 分类 + 跟踪
  ↓ 自动生成报告
```

## 数据库表（wechat_data.db）

| 表 | 用途 |
|------|------|
| `messages` | 所有微信消息 |
| `rooms` | 群配置 |
| `sync_cursors` | 增量同步游标 |
| `price_references` | 14 条历史报价 |
| `vehicle_types` | 7 条车型 |
| `contract_prices` | 合同价格 |
| `logistics_objects` | 物流对象（仅 2 条，需补充） |

## 分析能力

- **问题跟踪**: 135 个问题，每个有唯一 ID，从提出到解决全生命周期
- **决策记录**: 81 条
- **任务分工**: 自动提取
- **工作进展**: 自动提取
- **数据指标**: 订单量/时效/平台回复率
- **语言检测**: 中俄英自动分类
- **外部事件**: customs/rate 分类追踪
