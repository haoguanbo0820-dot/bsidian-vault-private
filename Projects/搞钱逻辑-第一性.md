# 搞钱逻辑 - 第一性原理

> 创建: 2026-06-30 (主人在 Hermes 内要我"学思路 学搞钱逻辑")
> 来源: 第一性推导 (不套 vault 现有项目, 不假设)

## 核心约束

钱 = 价值交换的凭证。
每一分钱 = A 给 B 价值, B 给 A 钱。
搞钱的本质 = 能给别人什么别人愿意付费的价值。

## 3 个最底层问题

### Q1: 钱从哪来?
- 主动掏钱的人, 不是你想赚就赚得到
- 谁主动掏钱? → 痛点 / 你掌握的信息 / 你控制的资源

### Q2: 钱怎么流到你这里?
1. 出卖时间 (打工接单, 天花板低)
2. 出卖产品 (卖书卖课, 边际成本低, 需流量)
3. 出卖资产 (收租利息股权, 需先有资产)
4. 出卖判断 (咨询投资, 高客单价, 需信任)

### Q3: 钱为什么给你 (不给你邻居)?
1. 信息差 (你比客户先知道, 易过期)
2. 信任差 (客户信你, 难建, 一旦建了稳)
3. 网络差 (你手里有客户连接, 网络效应)
4. 能力差 (你能做别人做不了, 可被 AI 削平)

## 4 铁律

1. **不要想钱** (想谁愿意付钱 + 为什么给你)
2. **不要泛做** (选 1 个细分人群, 做 1 件事做到极致)
3. **不要快** (信任差/网络差都需要时间)
4. **不要假** (数字你背书, 装逼的崩盘最快)
